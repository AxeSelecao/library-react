Тестовое задание: 
Реализовать web приложение для отслеживания библиотеки физических книг. 
Параметры книги: 
- автор
- название
- подумай какие еще нужны параметры для книги, чтобы обеспечить следующую функциональность

функциональность: 
1) в приложении пользователь должен иметь возможность:
- добавлять новую (свою) книгу
- добавить книгу, которую взял у кого-то
- удалять книгу
- отмечать как "дал почитать {$username}"
- "{$username} вернул" 
2) в приложении пользователь может просмотреть все имеющиеся у него книги и статусы по ним: "в библиотеке", "дал почитать"
3) в приложении должен быть реализован фильтр для книг по статусам: 
- пользователь может просмотреть все книги, которые "дал почитать"
- пользователь может просмотреть все книги, которые "взял почитать"

Реализовать web интерфейс на легком фреймворте типа bootstrap, язык бэкенда и базу выбери по своему усмотрению
